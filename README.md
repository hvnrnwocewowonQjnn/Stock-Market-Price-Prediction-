# 📈 Stock Market Price Prediction using LSTM

## About the Project

This project is a Stock Market Price Prediction system developed using **Deep Learning** with an **LSTM (Long Short-Term Memory)** network. The main objective of this project is to predict future stock closing prices based on historical market data.

I used the **Yahoo Finance API (yfinance)** to collect historical stock prices and trained an LSTM model to learn the price patterns over time. After training the model, I deployed it as a **Streamlit web application** where users can enter a stock ticker symbol and visualize the predicted stock prices along with historical trends.

---

## Features

- Fetch historical stock data using Yahoo Finance API
- Perform data preprocessing and normalization
- Train an LSTM model for time-series forecasting
- Predict future stock closing prices
- Display Moving Average (100 & 200 Days)
- Compare Actual vs Predicted Prices
- Interactive Streamlit Web Application

## Project Workflow

### 1. Data Collection

The historical stock data is collected using the **yfinance** library. The dataset contains Open, High, Low, Close and Volume information for the selected stock.

### 2. Data Preprocessing

The closing prices are extracted and normalized using **MinMaxScaler**. A sliding window approach is used to create training sequences where the previous 100 days are used to predict the next day's closing price.

### 3. Model Training

A multi-layer LSTM model is built using TensorFlow/Keras. The model is trained on historical stock prices using the Adam optimizer and Mean Squared Error (MSE) as the loss function.

### 4. Prediction

After training, the model predicts stock prices on the test dataset. The predicted values are converted back to the original scale and compared with the actual prices.

### 5. Model Saving

The trained model is saved as a **.keras** file so that it can be loaded later without retraining.

### 6. Streamlit Deployment

Finally, I developed a Streamlit web application where users can enter any stock ticker, fetch the latest market data, and visualize the predicted stock prices through interactive charts.
